* Consideração: Tempo Disponível

Restrições de tempo influenciam fortemente a escolha da técnica
#+attr_latex: :booktabs t
| *Critério*              | *Analítico*  | *Simulação*   | *Medição*         |
|                       | *Modelagem*  |             |                 |
|-----------------------+------------+-------------+-----------------|
| 2. Tempo necessário   | Pequeno    | Médio       | Variável        |

#+latex: \vfill

- Modelagem analítica
  - Adequada quando os resultados são necessários "para ontem"
- Simulações
  - Simulações são tipicamente lentas
- Medições
  - Mais longa que a modelagem analítica, mas mais curta que simulações
  - Duração altamente variável
    - Lei de Murphy: se algo pode dar errado, dará

* Consideração: Disponibilidade de Ferramentas

#+attr_latex: :booktabs t
| *Critério*              | *Analítico*  | *Simulação*   | *Medição*         |
|                       | *Modelagem*  |             |                 |
|-----------------------+------------+-------------+-----------------|

#+latex: \vfill

- Ferramentas necessárias
  - Modelagem analítica → habilidades de modelagem (ex.: teoria de filas)
  - Simulação → conhecimento de linguagens e técnicas de simulação
  - Medição → instrumentação

#+latex: \vfill

- Analistas diferem em expertise e preferências
  - Muitos modeladores experientes evitam sistemas reais
  - Outros preferem medição ou simulação por conhecimento limitado em modelagem
  - Falta de expertise em simulação frequentemente impede seu uso
#+latex: \pause
- As ferramentas necessárias para simulação podem não existir

\to A precisão é difícil de obter para qualquer técnica.

#+latex: \vfill

*A precisão varia por técnica*
  - Modelagem analítica → muitas suposições, frequentemente baixa precisão
  - Simulação → mais detalhes, mais próxima da realidade
  - Medição → pode variar de muito alta a zero (específica ao ambiente)

#+latex: \vfill\pause
  
Precisão é diferente de Correção das conclusões
- Risco de interpretação equivocada dos dados de desempenho

* Consideração: Avaliação de Trade-offs (/Insights/)

|                       | *Modelagem*  |             |                 |
|-----------------------+------------+-------------+-----------------|
| 5. Aval. de trade-off | Fácil      | Moderada    | Difícil         |

#+latex: \vfill

Estudo de desempenho
- Comparar alternativas
- Encontrar o valor ótimo de parâmetros

#+latex: \vfill\pause

- Modelos analíticos → melhores para entender efeitos de parâmetros
- Simulação → pode explorar o espaço de parâmetros, mas trade-offs pouco claros
- Medição → menos desejável para encontrar o valor ótimo de parâmetros
  - Resultados podem ser distorcidos por alguma mudança aleatória no ambiente

- Medições
  - A justificativa mais forte é a credibilidade
  - Mais fácil convencer outros com dados reais

#+latex: \vfill\pause

- Resultados analíticos
  - Frequentemente recebidos com ceticismo, pois métodos e resultados não são bem compreendidos
  - Prática comum
    - novas técnicas analíticas \to validar usando simulações ou medições reais

* Considerações Finais: Validando e Combinando Técnicas

#+begin_center
Até que se prove culpada, toda pessoa deve ser presumida inocente.

Até ser validado, todo resultado de avaliação é _suspeito_.
#+end_center
  
#+latex: \pause\vfill

Três regras de validação. Até ser validado por uma das outras duas técnicas
1. Não confie nos resultados de um modelo analítico
2. Não confie nos resultados de um modelo de simulação
3. Não confie nos resultados de medições (a maioria ignora esta regra)
#+latex: \pause

Frequentemente é útil usar duas ou mais técnicas em conjunto
- Simulação + Modelagem Analítica
- Medições + Simulação
#+latex: \pause\vfill

Medições também estão sujeitas a erros e bugs
- A intuição de especialistas é frequentemente usada como método de validação
  - Resultados não devem ser contraintuitivos

#+latex: \pause\vfill

Técnicas podem ser usadas sequencialmente
- Exemplo: modelo analítico define faixas de parâmetros → simulação refina
- Benefício: menos execuções de simulação, uso mais eficiente dos recursos
